## Requirements

### Device
- **Android** (Android 10+ recommended).
- Recommended free storage: **300 MB** (depends on final build size and assets).
- Internet connection: **not required** to play (only needed to download the APK).

### VR mode
- The VR prototype requires a device compatible with the VR setup used in this project.
- A modern/high-performance phone/tablet is recommended for smooth gameplay.

> Note: if you only play the 2D games, no special hardware is required.

## Install on Android

### Step 1 — Download the APK
1. Open the repository **Releases** section.
2. Download the `.apk` file to your Android device.

---

### Step 2 — Allow “Unknown sources”
Android may block installations that are not from the Play Store.

1. Open the downloaded APK.
2. Android will show a security warning.
3. Tap:
   - **Settings**
   - **Allow from this source**  
   (or “Allow unknown apps” depending on Android version)

> In some Android versions:
> Settings → Security → Install unknown apps

---

### Step 3 — Install
1. Open the `.apk` file.
2. Tap **Install**.
3. Wait until the installation finishes.

---

### Step 4 — Run
1. Open the app from your device menu.
2. Select game mode (Story Mode or Free Mode).
3. Start the mini-game you want to play.

---

## Permissions and recommendations

- If Android requests permissions (e.g., storage), grant them only if necessary.
- Recommended:
  - close heavy background apps
  - lower brightness if the device heats up
  - enable performance mode if available (depending on the device)

## VR notes

The repository includes a **VR prototype** (“Running of the Bulls”), inspired by San Fermín.

Recommendations:
- Ensure good lighting and enough physical space.
- Avoid sudden movements.
- Keep the device properly secured in the VR support/headset.
- If you feel motion sickness:
  - stop immediately
  - rest for a few minutes

## Common issues

### 1) The APK won’t install
- Confirm **unknown apps** installation is enabled.
- Make sure the download completed correctly.
- Restart the device and try again.

---

### 2) The app closes or shows a black screen
- This may happen due to low resources or device-specific issues.
- Recommended:
  - close background apps
  - restart the device
  - launch the app again

---

### 3) Touch controls don’t respond
- Check if your device has gesture navigation settings that interfere.
- Try again from the menu or restart the game.

---

### 4) VR performance issues (lag/stutter)
- Can occur on limited hardware.
- Recommendations:
  - close background apps
  - try a more powerful device
  - avoid long sessions (device temperature may affect performance)
